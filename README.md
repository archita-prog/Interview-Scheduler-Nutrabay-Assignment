# 📅 AI Interview Scheduling System

## Overview
I have created a AI-powered interview scheduling system that helps to identify the available and optimal interview slots between the candidates and the interviewrs.
It increases productivity and saves a lot of time by automatically overlapping availibilty and recommending best interview slots.
---

### Features
-Accepts candidate availibility(mutilple time slots supported)
-Handles multiple interviewers (max 5)
-Ranks top 3 best slots
-Higlights interviewer conficts
-Provides optimal recommendation

---
## Working 
1. The user inputs available slots for candidates and interviewers
2. The user can select multiple time slots also
3. After entering the data click on find slots button
4. As a result you can see ranks of slots based on:
   - Number of interviewers available for particular slot
   - Duration of overlap
   - Best options with reasoning
